## Descripción General
Este repositorio funciona como un portafolio digital donde se recopilan y organizan las prácticas de laboratorio y actividades de clase desarrolladas a lo largo del cuatrimestre. El propósito de este espacio es aplicar de manera práctica los conceptos teóricos de la minería de datos, abarcando desde la limpieza y preparación de los datos (Preprocesamiento), hasta el análisis exploratorio y la aplicación de algoritmos para descubrir patrones ocultos en bases de datos.
Cada práctica está desarrollada en un notebook independiente, documentando el flujo de trabajo completo del proceso de extracción de conocimiento.
